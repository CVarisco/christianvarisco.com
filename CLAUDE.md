# Christian Varisco Website

## Overview
Personal website for Christian Varisco, Fractional CTO. Built with Astro 5.x, Tailwind CSS v4, deployed on Vercel.

---

## CRITICAL: SEO & GEO Guidelines

**Every change must follow these SEO & GEO (Generative Engine Optimization) principles:**

### SEO Requirements
- All pages MUST have unique `<title>` and `<meta name="description">`
- Use semantic HTML: proper heading hierarchy (h1 > h2 > h3), article/section/nav
- Images MUST have descriptive `alt` text
- All links MUST have meaningful anchor text (no "click here")
- Maintain hreflang tags for multilingual pages
- Keep URLs clean and descriptive

### GEO/LLMO Requirements (AI Optimization)
- Update `/public/llms.txt` when adding significant content
- Maintain JSON-LD structured data in `/src/components/seo/JsonLd.astro` for:
  - Person schema (Christian's profile)
  - Organization schema (services)
  - FAQ schema (all FAQ items)
  - Service schema (each service offering)
- Use clear, factual language that AI can understand
- Structure content with clear headings and bullet points
- Include relevant keywords naturally in content

### When Adding New Content
1. Add JSON-LD schema if applicable
2. Sitemap is auto-generated by @astrojs/sitemap
3. Consider updating `/public/llms.txt` summary
4. Ensure proper heading hierarchy
5. Add alt text to all images

## Design System

### Colors
- **Accent (Brand)**: `#CBFC01` (lime green) - Use for CTAs, highlights, interactive elements
- **Background Primary**: `#000000` (black)
- **Background Secondary**: `#222222`
- **Text Primary**: `#FFFFFF` (white)
- **Text Secondary**: `#CCCCCC`

### Typography
- **Font**: Bricolage Grotesque (variable weight 200-800)
- Headings: weight 700, line-height 1.2
- Body: weight 400, line-height 1.5

### Signature Style
- **Solid Shadow**: `4px 4px 0 0 #CBFC01` for buttons and cards
- **Hover Effect**: Translate -2px and increase shadow to 6px

## i18n (Internationalization)

### URL Structure
- **English** (default): `https://christianvarisco.com/`
- **Italian**: `https://christianvarisco.com/it/`

### Adding/Editing Translations
1. Edit JSON files in `/src/data/en/` and `/src/data/it/`
2. Both language folders must have matching file names and JSON keys
3. Hreflang tags are handled automatically by `SEOHead.astro`

### Data Files
- `site.json` - SEO metadata, footer
- `navigation.json` - Nav links, CTA, social links
- `hero.json` - Hero section content
- `services.json` - Service offerings
- `results.json` - Stats and case studies
- `testimonials.json` - Client testimonials
- `timeline.json` - Process steps
- `faq.json` - FAQ items

## Adding Content

### Testimonials
Edit `/src/data/{lang}/testimonials.json`:
```json
{
  "items": [
    {
      "quote": "The testimonial text...",
      "author": "Name",
      "role": "Title at Company",
      "image": "/images/testimonials/name.jpg",
      "rating": 5
    }
  ]
}
```

### FAQ Items
Edit `/src/data/{lang}/faq.json` and update `/public/llms.txt`:
```json
{
  "items": [
    {
      "question": "Question text?",
      "answer": "Answer text..."
    }
  ]
}
```

### Case Studies (Future)
Add markdown files to `/src/content/case-studies/`
